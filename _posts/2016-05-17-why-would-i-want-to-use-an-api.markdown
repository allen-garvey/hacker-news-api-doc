---
title:  "Why would I want to use an API?"
date:   2016-05-16 17:01:19 -0400
---
APIs give you, the software developer, the power to use applications the way *you* want, rather than the other way around. Whether you want to create an alternative interface, automate common tasks, extend the application to new platforms or application analyze data, the only limits are those of the API itself.
